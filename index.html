Portafolio

Sistema Integral de Gestión Laboral

En esta organización surgió la necesidad de tener un sistema donde pudieran visualizar toda la información de los trabajadores de tal manera que a la hora de necesitar seleccionar una persona para un trabajo especifico poder ver sus cualidades e historial de posibles faltas que pudiese presentar ese trabajador, además de esto la unidad solicitante mantenía toda su gestión en papel físico o simplemente no lo tenían plasmado en ningún lado. Desarrolle un sistema que le permite al departamento tener una visual completa de la información de cada trabajador además de permitirles imprimir un récord laboral del trabajador en caso de necesitar, todo esto en conexión con otros sistema y con desarrollo nuevo, además de esto el sistemas permite al departamento llevar un control en linea de su gestión con posibilidad de adjuntar documentos, control de actuaciones diarias y estadísticas. Todo esto desarrollado con lenguajes como php, JavaScript, html, css y base de datos PostgreSQL.

Módulo de Control de Entrega de Medicinas a los Trabajadores

Para la organización solicitante la gestión de medicinas se hacia de manera manual con formularios físicos, proceso que era ineficiente debido a que dificultaba el seguimiento del historial de los trabajadores y sus familiares y estaba propenso a muchos errores de captación de datos. Desarrolle un modulo web que centraliza los registros, la gestión de inventario de medicamentos y la solicitud de medicinas por parte de los trabajadores adjuntando la documentación necesaria y esta disponible para todos los analistas que se encuentren en ese departamento al mismo tiempo. Este desarrollo se realizo bajo Php, JavaScript, html, css y con base de datos PostgreSQL.

Sistema de Gestión de Usuarios WiFi para Mikrotik

Este sistema surgió a partir de que antes los técnicos de la organización tenían que gestionar usuarios de forma manual en cada equipo Mikrotik, lo que consumía tiempo y estaba propenso a errores de transcripción de datos. Desarrolle un sistema web que centraliza la gestión de usuarios, utilizando tecnología de programación como Php, JavaScript, lenguaje de RouterOS y base de datos PostgreSQL. Con este sistema se logro agilizar el proceso, reducir errores y permitir la sincronizacion y monitoreo masivo de usuarios entre diferentes dispositivos
